# Lesson n°10









## Rule

The syllable before the suffix carries primary stress.

The vowel takes its orthographic value except <-i->, which is lax.


### Extending the rule

* Extended <-ion>: < e/i/u/y + V<sub>1</sub> + C <sub>0</sub> + (e) >
