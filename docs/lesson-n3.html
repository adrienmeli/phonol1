Cours de phonologie anglaise

3 Lesson n°3

3.1 Que savons-nous ?

3.1.1 Les consonnes

Transcrire tous les sons consonantiques, notamment ceux dont le symbole n’est pas un caractère d’imprimerie latin :

  • /ʃ/

  • /ʒ/

  • /θ/

  • /ð/

  • /ŋ/

3.1.2 Quelques voyelles

  • /ɪ/ comme dans “hit”.
  • /e/ comme dans “bed”.
  • /eɪ/ comme dans “hate”.
  • /aɪ/ comme dans “site”.
  • /aʊ/ comme dans “now”.

3.4 Homework

11/10/2021 Trouver dans un dictionnaire la transcription des mots suivants : “hat“, “pet“, “sit“, “not“, “cut“, “put“. Pour chacune de ces voyelles, trouver 5 autres mots comportant au moins une syllabe présentant ces voyelles.
