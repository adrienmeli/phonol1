# Lesson n°3



## Que savons-nous ?

### Les consonnes

Transcrire tous les sons consonantiques, notamment ceux dont le symbole n'est pas un caractère d'imprimerie latin :

* /ʃ/

* /ʒ/

* /θ/

* /ð/

* /ŋ/

---

### Quelques voyelles

*  /ɪ/ comme dans "hit".
*  /e/ comme dans "bed".
*  /eɪ/ comme dans "hate".
*  /aɪ/ comme dans "site".
*  /aʊ/ comme dans "now".
